# Exam for capitolis company



## contents
 - react 15.5+
 - [Material UI](http://www.material-ui.com/)
 - react-router v4
 - Roboto font
 - [Flexbox Grid](http://flexboxgrid.com/)
 - react-hot-loader v3+
 - webpack 2.x

## Usage

### Install

```bash
npm install
```
or 

```bash
yarn install
```

### Run

```bash
npm start
```
or 

```bash
yarn start
```
